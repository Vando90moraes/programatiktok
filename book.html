Chapter 1 · Tools

Your Essential Creator Stack

Before recording any video, set up these 5 tools. They form the minimum operational kit to run a professional beauty creator business.

1. Dolphin Anty — Multi-Account Management

An anti-detect browser that simulates unique device profiles. Each TikTok account runs in a separate "virtual computer" — the algorithm sees completely different users, not accounts linked to the same person.

  • Manage 5+ TikTok accounts simultaneously
  • Each profile has isolated IP, browser fingerprint, and cookies
  • Prevents TikTok from linking your accounts to each other
  • Can automate repetitive actions using Selenium or Puppeteer

2. ChatGPT — Scripts, Captions, and SEO

Use GPT-4 to generate video scripts, create hook variations, optimize descriptions for SEO, and build complete content calendars. The key is knowing exactly which prompts to give it — Chapter 3 covers these in detail.

  • Generate 10 hook variations for any product in seconds
  • Create optimized 30s/60s scripts in your voice
  • Suggest hashtags by niche and platform
  • Adapt the same content for each social network

3. Descript — Text-Based Video Editing

Auto-transcribes your video and lets you edit it by deleting words — like editing a Word document. Generates captions in seconds and removes filler words with one click.

  • Auto-transcription in English (and other languages)
  • Remove "uh," "um," and silences with one click
  • Export a clean video file to import into CapCut
  • Generate SRT caption files for YouTube

4. CapCut — Finishing and Publishing

Mobile and desktop video editor natively integrated with TikTok. Apply trending effects, animated text, beauty filters, and publish directly to TikTok without leaving the app.

  • Viral beauty templates ready to use immediately
  • Automatic skin smoothing and color effects
  • Import video from Descript and finish here
  • Publish directly to TikTok and Instagram Reels

5. LTK (LikeToKnow.it) + Impact.com

LTK centralizes your affiliate links in a single visual storefront. Impact.com is the platform where TULA, CapCut, Ulta, and other major programs live. Set up both before you start posting — you'll need them ready when your first video goes up.

  • LTK: quick approval, works with any beauty product
  • Impact.com: home of the biggest beauty niche programs (TULA, Alpyn, CapCut, Ulta)
  • One LTK link holds multiple products with individual tracking
  • Detailed reports on clicks, conversions, and commissions per product
Setup Order Week 1: Install Dolphin Anty, CapCut, and Descript. Create accounts on Impact.com, ShareASale, and LTK. These registrations run in the background while you build your first content.

Chapter 2 · Dolphin Anty

Multi-Account Without the Ban

Step-by-step guide to setting up multiple isolated profiles. Each account operates as if it were a different physical device.

Why This Matters TikTok detects multi-accounts through browser fingerprint, IP address, and shared cookies. Dolphin Anty creates completely isolated digital identities — TikTok cannot link your accounts to each other.
1
Install Dolphin Anty and Create an Account

Go to dolphin-anty.com → download for Mac or Windows. Create a free account (free plan: up to 10 profiles). For more accounts, consider the Basic plan ($89/mo) or Team ($159/mo).

  • Install and log in to the web dashboard
  • Set the language to English for easier navigation
  • Under the Profiles tab you will manage all your browser profiles
Tip: The free plan is sufficient to test with 2–3 accounts before committing to a paid plan.
2
Set Up Proxies — One Per Account

Each profile must have a different proxy. Use residential proxies (more reliable than datacenter proxies for TikTok detection avoidance).

  • Recommended services: Bright Data, Oxylabs, IPRoyal
  • Type: Residential (not datacenter)
  • Location: match your actual city
  • In Dolphin → Profile → Proxy → paste: host:port:user:password
  • Click Check Proxy — the correct IP should appear
Tip: Mobile proxies are the hardest for TikTok to detect if your budget allows.
3
Create and Configure Browser Profiles

Each profile is a unique digital identity. Configure each one as follows:

  • Name: use the account's niche (e.g., "Skincare 35+ Account")
  • OS: vary between Windows 10 and macOS across profiles
  • Browser: Chrome (recommended for TikTok)
  • Screen Resolution: alternate between 1920×1080 and 1366×768
  • Language: English (US) with timezone America/Chicago
  • WebGL/Canvas: set to "Noise" — generates a unique fingerprint per profile
Golden Rule: Never open two profiles from the same niche at the same time. Alternate with at least 30 minutes between sessions.
4
Create a TikTok Account Inside the Isolated Profile

With the Dolphin profile open, create the TikTok account as if it were a brand new device:

  • Go to tiktok.com inside the Dolphin profile
  • Use a different email for each profile — Gmail aliases work: you+account2@gmail.com
  • Use a different phone number (or a virtual SMS service like TextNow)
  • Spend the first 3 days consuming niche content only — do not post yet
  • Enable two-factor authentication (2FA) on each account
Warm-Up Period: New accounts that post immediately are flagged as suspicious. Consume 30–60 minutes of content per day for 3 days before your first post.
5
Daily Operation Rules

To keep accounts safe long-term, follow these rules every day:

  • Never copy and paste the same video across accounts — TikTok detects duplicate metadata
  • Each account must have a different niche (skincare, makeup, clean beauty, etc.)
  • Never access an account outside its correct Dolphin profile
  • Save sessions in Dolphin after each use (the "Save Session" button)
  • If an account gets banned, change the proxy first before attempting any recovery

Chapter 3 · ChatGPT

ChatGPT as Your Content Co-Creator

Don't use GPT just to write text. Use it as a co-creator at every stage — from scripting to SEO optimization.

Prompt 1 — Generate Hooks for Any Product

"Give me 10 opening hooks (first 3 seconds) for a TikTok video about [product name]. Each hook should create curiosity, urgency, or envy. Target audience: women 25–45. Tone: honest, not salesy."

Use the hook that feels most natural to you — you are the one recording, so it needs to sound authentic. Test 3 different hooks across different videos and compare watch time in TikTok Analytics.

Prompt 2 — Create a 30–60 Second Script

"Write a 45-second TikTok script for [product]. Structure: Hook (3s) → Demo/context (20s) → Concrete result (15s) → Simple CTA (5s). Conversational tone, first person, doesn't sound like an ad. Include where to pause and where to show the product on camera."

Prompt 3 — Optimized Captions and Descriptions

After recording, use GPT to adapt the script into captions for each platform:

"Based on this script: [paste script], create: 1) TikTok caption (150 characters + 5 beauty niche hashtags), 2) Instagram Reels caption (300 characters + 3 hashtags), 3) YouTube Shorts title (60 characters, with main keyword), 4) YouTube description (150 characters with natural keywords)."

Prompt 4 — Weekly Content Calendar

"Create a 7-day content calendar for a beauty TikTok account focused on skincare and affiliates (Ulta, TULA, Amazon). For each day: video type, suggested hook, focal product, which affiliate program to promote, and best posting time. Maximum 2 videos per day."
Using GPT Effectively GPT's first draft is rarely perfect. Give it a prompt, read the result, then add: "Make it shorter," "Make the hook more provocative," or "Write it in a more casual voice." The second or third iteration is usually what you will actually record.

Chapter 4 · Descript

Produce Professional Video in Descript

Descript turns video editing into text editing. You record, it transcribes, you delete words — the video cuts automatically with the text.

The Workflow

Record (smartphone or webcam) → Import into Descript → Transcribe (auto, seconds) → Edit text (delete words = cut video) → Export MP4 for CapCut

1
Set Up a Project in Descript
  • Download Descript at descript.com — free plan includes 1 hour/month; Creator plan is $24/mo for heavy use
  • Create a new Workspace (e.g., "Beauty Creator")
  • Click New Project → Video Project
  • Set transcription language to English
2
Import Your Recorded Video
  • Drag your video file (MP4, MOV, any format) into Descript's timeline
  • Wait for auto-transcription — typically 1–2 minutes per minute of video
  • Review the transcript and click any word to correct it
  • The text appears side by side with the video on the timeline
Recording Tip: Always record in 1080p with good lighting. Speak at a steady pace — transcription accuracy is higher and auto-cuts are cleaner.
3
Edit by Cutting Text
  • Select unnecessary words or phrases → Delete → the video cuts automatically
  • Use Remove Filler Words (Edit menu) to delete all "uh," "um," and "like" at once
  • Use Remove Silences to cut long pauses automatically
  • To reorder scenes: select a text block → cut and paste it to the new position
Pro Trick: Duplicate a project to create 15s, 30s, and 60s versions of the same video. Export three files from one recording session.
4
Export for CapCut
  • Click Export → Video
  • Resolution: 1080×1920 (vertical 9:16 for TikTok and Reels)
  • Format: MP4 H.264
  • Do not export with burned-in captions yet — CapCut adds styled captions
  • Save with a descriptive filename: tula-serum-review-clean.mp4
Optional: Export the SRT file (captions) via File → Export → Transcript → SRT for use on YouTube.

Chapter 5 · CapCut

Finish and Publish in CapCut

CapCut is where the video comes to life visually. Import the clean MP4 from Descript and add captions, music, effects, and publish.

1
Import the Video from Descript
  • Open CapCut on mobile or desktop (capcut.com)
  • Tap + New Project → select the MP4 exported from Descript
  • Confirm the ratio: 9:16 (vertical) for TikTok/Reels
  • If recorded horizontally, use Auto Reframe — it detects your face and crops automatically
2
Add Auto-Captions
  • In the timeline, tap Text → Auto Captions
  • Select language → click Start (takes about 30 seconds)
  • For beauty content: use bold white fonts with black shadow or solid dark background
  • Recommended position: center-bottom — does not cover your face
  • Review and correct any product names that were mistranscribed
Viral Style: Captions with word-by-word color animation increase watch time by approximately 15%.
3
Add Trending Music or Sound
  • Tap Audio → Sounds → filter by TikTok Trending
  • Use a sound from the top 20 in the beauty niche
  • Music volume: 10–20% — your voice is the primary audio
  • For honest review videos, no music at all often converts better
  • For B-roll product videos without voiceover, music can go up to 50–70%
4
Beauty Effects and Color Grading
  • Color: Filters → "Skin" category → subtle intensity (30–50%)
  • Smoothing: Effects → Beauty → Skin Smoothing — use at 30% maximum to stay natural
  • LUT: a neutral or warm filter gives a premium look for skincare content
  • Avoid filters that warp your face or erase pores — authenticity is your product
5
Export and Publish
  • Click Export → Resolution: 1080p, Frame rate: 30fps
  • Use the Post to TikTok option inside CapCut — faster than saving and re-uploading
  • For Instagram Reels: export and upload manually in Instagram
  • For YouTube Shorts: same file, upload via YouTube Studio
  • Important: do not repost the exact same file to multiple platforms without changes — TikTok penalizes content with other platforms' watermarks
Hack: Enable "Remove Watermark" in CapCut before exporting to use the video on Instagram without the CapCut logo.

Chapter 6 · Growth

Grow to 10K–30K Followers Organically

Without followers, there are no commissions. This chapter covers the real levers of organic growth on TikTok based on algorithm behavior data.

1
Optimize Your Profile Before Posting
  • Photo: clear face, smiling, good lighting — not a logo or product shot
  • Username: include your niche (@beautyfinds_nash, @skincarehonest)
  • Bio (80 chars): niche + benefit + CTA. Example: "Honest Skincare 🔬 | Ulta + Amazon Finds | Link ↓"
  • Link in bio: use LTK or Linktree with your affiliate links organized by category
  • Connect your Instagram and YouTube to your TikTok profile
2
Own One Niche Per Account

With multiple accounts via Dolphin Anty, each must have a distinct niche:

  • Account 1: Skincare 35+ — TULA, Ulta, anti-aging routines
  • Account 2: Amazon beauty finds — products under $30 with high search volume
  • Account 3: Clean Beauty / Natural — Alpyn Beauty, sustainable brands
  • Account 4: Makeup for deep/dark skin — Sigma Beauty, inclusivity focus
  • Account 5: Haircare — less saturated than skincare
Rule: Stay in one niche for 30 days without deviating. The algorithm takes 2–3 weeks to understand your content and route it to the right audience.
3
Posting Cadence and Best Times
  • Minimum: 1 video per day for the first 30 days
  • Ideal: 2 videos per day at different times
  • Morning: 7am–9am (commuting / coffee)
  • Lunch: 12pm–1pm (work break)
  • Evening: 7pm–9pm (highest usage volume)
  • Use TikTok Analytics → Followers → Activity to confirm when your specific audience is online
4
Engagement in the First 30 Minutes

TikTok's algorithm evaluates early engagement to decide how widely to distribute a video. In the first 30 minutes after posting:

  • Reply to every comment — even a single emoji counts
  • Pin one of your own comments with a question to drive more replies
  • Share the video to your Instagram Stories
  • Do not edit the caption after posting — TikTok resets distribution when you do
5
Duets, Stitch, and Trending Sounds
  • Stitch: take viral videos from your niche (10M+ views) and add your opinion — you piggyback on their distribution
  • Duet: great for reacting to product reviews you also use
  • Trending sounds: check the "Trends" tab in Creator Center — use audios with a green upward arrow ↑
  • Trending sounds give automatic organic boost even to small accounts
  • Limit Stitches to 2–3 per week — do not replace original content with them

Chapter 7 · TikTok Shop

Register and Monetize on TikTok Shop

TikTok Shop is the fastest available monetization path. Purchases happen inside the app — conversion rates are 3–5× higher than external links.

Minimum Requirement: 1,000 Followers Focus on organic growth (Chapter 6) until you reach 1K. With a well-optimized account posting daily, this typically takes 2–4 weeks.
1
Register as an Affiliate in Creator Center
  • Go to creator.tiktok.com → log in with your TikTok account
  • Sidebar → TikTok Shop → Affiliate
  • Click Apply Now → fill in: name, address, bank details
  • Approval usually takes 1–3 business days
  • After approval, access Product Marketplace to browse products
2
Choose Products to Promote
  • Filter by: Category: Beauty, Commission: 10%+, Sales: 100+/month
  • Prioritize products with 4.5★ or higher — they have existing social proof
  • Check if Ulta brands are available on TikTok Shop before creating content
  • Save favorites to My Selections for quick access
Strategy: Pick 3–5 fixed products and run them for 30 days before switching. Consistency builds product association in your audience's mind.
3
Pin the Product to Your Video
  • On TikTok's publish screen, scroll to Add Link
  • Select TikTok Shop Product → search the product by name
  • It appears as an orange button on the video — viewers tap and buy without leaving TikTok
  • Mention it out loud: "tap the product below" or "link is right on the video"
4
Track Commissions and Payment
  • Creator Center → TikTok Shop → My Commissions
  • View: product clicks, orders, conversion rate, amount pending
  • Payment: weekly, directly to your registered bank account
  • Holding period: commission is confirmed 7 days after product delivery to the customer

Chapter 9 · Content Structure

The Content Formula That Converts

Every successful affiliate video follows the same 4-act structure. No exceptions.

01
HOOK — 0 to 3 Seconds — The Question That Stops the Scroll

If it does not stop the scroll in 3 seconds, the video does not exist for that viewer. Start with conflict, not with an introduction.

  • "Is this $90 serum actually worth it?" (creates doubt)
  • "I tested the most expensive Sephora product for 30 days…" (builds suspense)
  • "Why is everyone buying this at Ulta right now?" (triggers FOMO)
  • "This product wrecked my skin… in the best way." (unexpected twist)
Rule: Never start with "Hey, how's it going?" or by introducing yourself. The audience wants to know immediately what you have for them — not who you are.
02
DEMO — 3 to 25 Seconds — Real Demonstration

Show application on skin, texture, scent (describe it verbally), consistency. No heavy cuts. No extreme filters. Authenticity is the product.

  • Before / after in one continuous clip
  • Zoom in on texture and pigmentation
  • Show a genuine, real-time reaction
  • Talk about what you feel, not what the brand's marketing says
03
PROOF — 25 to 45 Seconds — Concrete Result

Numbers, comparisons, a close-up skin shot, an honest opinion. If it did not work 100%, say so — honesty triples credibility and long-term conversions.

  • "After 7 days, visibly smaller pores."
  • "Lasted 8 hours without a touch-up — I tested it at work."
  • "I didn't love the packaging, but the product itself…"
  • Mention price and where to buy at this point in the video
04
CTA — Last 3 Seconds — One Action, Only One

Two CTAs equal confusion and zero clicks. Choose one platform, one action, one link per video.

  • "Link in bio — buy directly at Ulta." (Ulta affiliate)
  • "Tap the product below." (TikTok Shop)
  • "Check my LTK — it's in the links." (LTK storefront)
  • "Comment WANT and I'll send you the link." (engagement + link in comments)
Why "Comment WANT" Works: Comments trigger the algorithm to distribute more → more people see the video → more potential sales. Use this CTA 2–3 times per week, not every video.

Chapter 10 · SEO

SEO by Platform

Each platform has its own search algorithm. The same content must be optimized differently for each one.

TikTok SEO

  • Spoken keywords: say the product name and category out loud in the first 10 seconds — TikTok transcribes audio and indexes it for search
  • On-screen text: put visible keywords in the video (e.g., "TULA Skincare Review")
  • Caption: first sentence with main keyword as plain text — not a hashtag
  • Hashtags: 3–5 only — 1 broad (#beauty), 1 niche (#skincaeroutine), 1 specific (#tulareviews)
  • Trending sound: top-trending audios receive automatic preferential distribution
  • Timing: post when your audience is most active (Analytics → Followers → Activity)

YouTube SEO

  • Title (60 chars max): main keyword + benefit. Example: "TULA Skincare Honest Review — 30 Days Results"
  • Description: first 150 characters are indexed — put keyword, product name, and affiliate link here
  • Tags: 10–15 tags mixing broad ("skincare review") and specific ("tula cleanser review 2026")
  • Chapters: add timestamps with keywords — they appear as links in Google search results
  • Thumbnail: your face + high contrast + 3 words maximum (e.g., "WORTH IT?")
  • Shorts: keyword in title + #Shorts in description + affiliate link in first 3 description links

Instagram SEO

  • Display name: include a keyword in your display name (e.g., "Maria | Skincare Reviews") — it appears in Instagram search results
  • Caption: Instagram indexes caption text — write naturally but include product name and category
  • Alt text: on every post, edit the image alt text with descriptive keywords
  • Hashtags: 3–5 (the current algorithm penalizes the 20–30 hashtag approach)
  • Reels description: first sentence with keyword — appears in Instagram Search
  • Location: geotag your city for local and national audience reach

Facebook SEO

  • Groups: highest organic reach on Facebook — join beauty groups and share your videos with context
  • Native upload: always upload videos directly to Facebook, never as YouTube links — the algorithm strongly favors native content
  • Video title: treat it like a blog headline — keyword + benefit + action
  • Description: the first 3 lines appear without clicking "see more" — put your keywords here
  • Page vs. Profile: create a Creator Page for access to Facebook Insights and greater organic reach
Universal Cross-Platform Rule Same product, described in different ways, with keywords adapted for each platform's algorithm. Never copy and paste the identical caption to TikTok, Instagram, YouTube, and Facebook — each platform maintains its own search index.

Chapter 11 · Execution

Your 90-Day Plan

Exact sequence of actions organized by week. Do not skip phases — each one sets up the next.

1

Week 1 — Complete Setup

Before creating any content, configure your full infrastructure.

  • Install Dolphin Anty and configure 2–3 profiles with residential proxies
  • Create TikTok accounts inside Dolphin profiles — warm up for 3 days (consume, do not post)
  • Download CapCut (mobile + desktop) and Descript — create test projects to learn the workflow
  • Create accounts on Impact.com, ShareASale, and LTK
  • Apply for: TikTok Shop Affiliate, LTK, and Amazon Associates
  • Define the niche of each account — create optimized bios with keywords
  • Set up your link in bio (LTK or Linktree) with affiliate links organized and ready
2

Week 2 — First Videos + New Affiliates

Consistent posting begins here. Apply the Hook → Demo → Proof → CTA formula to every video.

  • Post 1–2 videos per day on each active account with a TikTok Shop product pinned
  • Full workflow: record on smartphone → edit in Descript → finish in CapCut → publish
  • Apply for TULA Skincare (Impact.com) and Alpyn Beauty (ShareASale)
  • Use ChatGPT to generate hooks and scripts — test at least 3 types of openings per week
  • Reply to every comment within the first 30 minutes after each post
  • Do 2 Stitches of viral niche videos during the week
3

Week 3 — Ulta + CapCut Affiliate

Add the slower-approval programs while your content is already running.

  • Apply for Ulta UBCreates (Rakuten / Impact) — the process can take 1–2 weeks to complete
  • Apply for CapCut Pro affiliate (Impact.com) — create 1 "how I edit my videos" tutorial
  • Analyze early data: which hook type generated the most watch time? Replicate it
  • Distribute your top-performing videos to Instagram Reels and YouTube Shorts
  • Apply SEO optimization for each platform following Chapter 10
4

Weeks 4–8 — Consistency and Optimization

The learning phase. Data from weeks 1–3 will tell you exactly what works.

  • Identify the 3 video formats that converted best → double production of those formats
  • Pause formats that produced no results after 10+ videos
  • Test 1 new product per week on TikTok Shop — retain only the ones that generate sales
  • Reach 1,000 followers on at least 1 account — required for TikTok Shop activation
  • Start a recurring content series ("Weekly Skincare," "Find of the Week") — builds audience loyalty
  • Track conversion data per affiliate program — cut programs that do not convert
5

Month 2–3 — Scale What Works

Do not scale before you have data. With 30–60 days of results, you know exactly what to amplify.

  • Apply for Sigma Beauty PRO (ShareASale) on the makeup-focused account when it has traction
  • Increase posting frequency in the best-performing niches
  • Outsource parts of the workflow: thumbnails, basic editing, content calendar (Fiverr freelancers)
  • Negotiate direct brand deals with brands you are already promoting organically
  • Reinvest first commissions into better proxies and additional Dolphin profiles
  • Goal: first recurring passive income from subscription-based affiliates (TULA subscriptions, CapCut Pro)
