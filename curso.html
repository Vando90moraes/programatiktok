Module 2 · Dolphin Anty

TikTok Shop Multi-Account Without Ban

Step-by-step guide to setting up multiple isolated profiles. Each account operates as if it were a different physical device.

⚠️
Why do this? TikTok detects multi-accounts via browser fingerprint, IP, and shared cookies. Dolphin Anty creates completely isolated digital identities — TikTok cannot link your accounts to each other.
1
Install Dolphin Anty and create an account ▼

Go to dolphin-anty.com → download for Mac or Windows. Create a free account (free plan: up to 10 profiles). For more accounts, Basic plan ($89/mo) or Team ($159/mo).

  • Install and log in to the web dashboard
  • Set the language to English for easier navigation
  • Under the Profiles tab, you'll see all your browser profiles
Tip: The free plan is enough to test with 2–3 accounts before investing in a paid plan.
2
Set up proxies — one per account ▼

Each profile MUST have a different proxy. Use residential proxies (more reliable than datacenter for TikTok).

  • Recommended services: Bright Data, Oxylabs, IPRoyal
  • Type: Residential (not datacenter)
  • Location: Nashville, TN (match your actual city)
  • In Dolphin → Profile → Proxy → paste proxy data: host:port:user:password
  • Click Check Proxy — the correct IP should appear
Tip: Use mobile proxies if budget allows — they are the hardest for TikTok to detect.
3
Create and configure browser profiles ▼

Each profile = a unique digital identity. Configure it like this:

  • Name: use the account name (e.g. "Skincare 35+ Account")
  • OS: Windows 10 or macOS (vary between profiles)
  • Browser: Chrome (recommended for TikTok)
  • Screen Resolution: vary between 1920x1080, 1366x768
  • Language: English (US) with timezone America/Chicago
  • WebGL/Canvas: set to "Noise" (generates a unique fingerprint)
Golden rule: Never open two profiles in the same niche at the same time. Alternate with at least 30 minutes between sessions.
4
Create a TikTok account inside the isolated profile ▼

With the Dolphin profile open, create the TikTok account as if it were a brand new device:

  • Go to tiktok.com inside the Dolphin profile
  • Create the account with a different email for each profile (use Gmail aliases: youremail+account2@gmail.com)
  • Use a different phone number (or a virtual SMS service like TextNow)
  • Spend the first 3 days only consuming niche content — don't post yet
  • Enable 2FA (two-factor authentication) on each account
Warm-up period: New accounts that post immediately are more suspicious. Consume 30–60 min of content per day for 3 days before posting.
5
Daily operation rules ▼

To keep accounts safe long-term:

  • Never copy/paste the same video across accounts — TikTok detects duplicate metadata
  • Each account must have a different niche (skincare, makeup, clean beauty, etc.)
  • Never access an account outside its correct Dolphin profile
  • Save sessions in Dolphin after each use ("Save Session" button)
  • If an account gets banned, don't try to recover it with the same proxy — change the proxy first

Module 3 · ChatGPT

ChatGPT in Your Creation Workflow

Don't use GPT just to "write a text." Use it as a co-creator at every step of the process — from scripting to SEO.

1
Generate hooks for any product ▼

Exact prompt to use:

"Give me 10 opening hooks (first 3 seconds) for a TikTok video about [product name]. The hook should create curiosity, urgency, or envy. Audience: women 25–45 years old. Tone: honest, not salesy."

Use the hook that feels most natural to you — you're the one recording, so it needs to sound authentic.

2
Create a 30–60 second script ▼

Prompt for a full script:

"Write a 45-second TikTok script for [product]. Structure: Hook (3s) → Demo/context (20s) → Concrete result (15s) → Simple CTA (5s). Conversational tone, first person, doesn't sound like an ad. Include where to pause and where to show the product on camera."
3
Generate optimized captions and descriptions ▼

After recording, ask GPT to adapt the script into captions for each platform:

"Based on this script: [paste script], create: 1) TikTok caption (150 chars + 5 beauty niche hashtags) 2) Instagram Reels caption (300 chars + 3 hashtags) 3) YouTube Shorts title (60 chars, with main keyword) 4) YouTube description (150 chars with natural keywords)"
4
Weekly content calendar ▼
"Create a 7-day content calendar for a beauty TikTok account focused on skincare and affiliates (Ulta, TULA, Amazon). I want: video type, suggested hook, focal product, which affiliate program to promote, best posting time. Max 2 videos per day."

Set up a project in Descript ▼
  • Download Descript at descript.com (free plan: 1hr/month free; Creator $24/mo for heavy use)
  • Create a new Workspace (e.g. "Beauty Creator")
  • Click New Project → Video Project
  • Set transcription language: English if recording in English
2
Import your recorded video ▼
  • Drag your video file into Descript's timeline (MP4, MOV, any format)
  • Wait for auto-transcription (usually 1–2 min per minute of video)
  • Review the transcript — click on any word to correct it
  • The text appears side by side with the video on the timeline
Recording tip: Always record in 1080p with good lighting. Speak at a steady pace — transcription is more accurate and auto-cuts are cleaner.
3
Edit by cutting text ▼
  • Select unnecessary words/phrases → Delete → the video cuts automatically
  • Use Remove Filler Words (Edit menu) to delete all "uh", "um", "like" at once
  • Use Remove Silences to cut long pauses automatically
  • To reorder scenes: select a text block → cut and paste it elsewhere
  • Add Scenes (chapters) to organize the structure visually
Pro trick: Use Descript to create 15s, 30s, and 60s versions of the same video — duplicate the project, cut each to length, export three separate files.
4
Export for CapCut ▼
  • Click Export → Video
  • Resolution: 1080x1920 (vertical 9:16 for TikTok/Reels)
  • Format: MP4 H.264
  • Do not export with burned-in captions yet — CapCut will add styled captions
  • Save to your project folder with a descriptive name: tula-serum-review-clean.mp4
Optional: Also export the SRT file (captions) via File → Export → Transcript → SRT to use on YouTube.

Module 5 · CapCut

Finish and Publish in CapCut

CapCut is where the video comes to life visually. Import the clean MP4 from Descript and add the elements that drive engagement: captions, text, music, and effects.

1
Import the video from Descript ▼
  • Open CapCut on mobile or desktop (capcut.com)
  • Tap + New Project → select the MP4 exported from Descript
  • Confirm the ratio: 9:16 (vertical) for TikTok/Reels
  • If you recorded horizontally, use Auto Reframe — it detects your face and crops automatically
2
Add auto-captions ▼
  • In the timeline, tap Text → Auto Captions
  • Select language → click Start (takes ~30 seconds)
  • Choose the style: for beauty, use bold white fonts with black shadow or solid dark background
  • Recommended position: center-bottom (doesn't cover your face; stays above TikTok buttons)
  • Review and correct any product names that were mistranscribed
Viral style: Captions that change color on keywords (use "Word-by-word text animation") increase watch time by ~15%.
3
Add trending music or sound ▼
  • Tap Audio → Sounds
  • Filter by TikTok Trending — use a sound from the top 20 in the beauty niche
  • Music volume: 10–20% — your voice is the main audio, music is background
  • If your voice is clear, consider no music at all — honest review videos convert well without it
  • For product videos without voiceover (B-roll), music can go up to 50–70%
4
Beauty effects and color grading ▼
  • Color adjustment: Filters → "Skin" category → choose a subtle one (intensity 30–50%)
  • Smoothing: Effects → Beauty → Skin Smoothing — use sparingly (30% max) to look natural
  • Cinematic LUT: For a premium look, apply a neutral/warm filter — works well for skincare
  • Never use filters that warp your face or completely erase pores — it kills authenticity
5
Export and publish ▼
  • Click Export → Resolution: 1080p, Frame rate: 30fps
  • Use the Post to TikTok option inside CapCut (faster than saving and re-uploading)
  • For Instagram Reels: export the file and upload manually in Instagram
  • For YouTube Shorts: same file, upload via YouTube Studio
  • Do not repost the exact same file on TikTok and Instagram without at least one change (re-exporting is enough) — TikTok penalizes content with other platforms' watermarks
Hack: In CapCut, enable "Remove watermark" before exporting to use the video on Instagram without the CapCut logo.

Module 6 · Growth

Grow to 10K–30K Followers

Without followers, there are no commissions. This module covers the real levers of organic growth on TikTok — based on algorithm behavior data.

1
Optimize your profile before posting ▼
  • Photo: Clear face, smiling, good lighting — don't use a logo or product
  • Username: include your niche (@beautyfinds_nash, @skincarehonest)
  • Bio (80 chars): niche + benefit + CTA. Ex: "Honest Skincare 🔬 | Ulta + Amazon Finds | Link ↓"
  • Link in bio: use LTK or Linktree with your affiliate links organized
  • Connect Instagram and YouTube to your TikTok profile
2
Pick and own one niche per account ▼

With multiple accounts (Dolphin Anty), each must have a clear niche. Suggestions:

  • Account 1: Skincare 35+ — TULA, Ulta, anti-aging routines
  • Account 2: Amazon beauty finds — products under $30 with high volume
  • Account 3: Clean Beauty / Natural — Alpyn Beauty, sustainable brands
  • Account 4: Makeup for deep/dark skin — Sigma Beauty, inclusivity
  • Account 5: Haircare — less saturated niche than skincare
Rule: Stay in 1 niche for 30 days without deviating. The algorithm takes 2–3 weeks to understand what you post and start distributing to the right audience.
3
Posting cadence and best times ▼
  • Minimum frequency: 1 video per day for the first 30 days
  • Ideal: 2 videos per day at different times
  • Best posting times (Nashville, ET):
  • Morning: 7am–9am (people commuting or having coffee)
  • Lunch: 12pm–1pm (work break)
  • Evening: 7pm–9pm (highest usage volume)
  • Test and use TikTok Analytics to confirm when your specific audience is active
4
Engagement in the first 30 minutes ▼

TikTok's algorithm evaluates early performance to decide reach. In the first 30 min after posting:

  • Reply to every comment — even with just an emoji
  • Pin 1 comment of yours with a question (drives more replies)
  • Share the video to your Instagram Stories
  • Don't edit the caption after posting — TikTok resets distribution
5
Duets, Stitch, and trending sounds ▼
  • Stitch: Take viral videos from your niche (10M+ views) and add your opinion — you piggyback on their traffic
  • Duet: Great for reacting to product reviews you also use
  • Trending sounds: Always check the "Trends" tab in Creator Center — use audios with a green upward arrow ↑
  • Trending sounds give automatic organic boost even to small accounts
  • Limit: 2–3 Stitches per week — don't replace original content

Module 7 · TikTok Shop

Register and Monetize on TikTok Shop

TikTok Shop is the fastest monetization available. Purchases happen inside the app — 3–5x higher conversion rate than external links.

💡
Minimum requirement: 1,000 followers Focus on organic growth (Module 6) until you hit 1K. With a well-optimized account posting daily, this takes 2–4 weeks.
1
Register as an affiliate in Creator Center ▼
  • Go to creator.tiktok.com → log in with your TikTok account
  • Sidebar → TikTok Shop → Affiliate
  • Click Apply Now → fill in: name, address (Nashville), bank details
  • Approval usually takes 1–3 business days
  • After approval, access Product Marketplace to explore products
2
Choose products to promote ▼
  • In Product Marketplace, filter by: Category: Beauty, Commission: 10%+, Sales: 100+/month
  • Prioritize products with many positive reviews (4.5★+) — they already have social proof
  • Check if Ulta brands are available on TikTok Shop before creating content
  • Save favorite products to My Selections for quick access
  • Commission varies by category: skincare pays more than accessories
Strategy: Pick 3–5 fixed products to run for 30 days instead of switching every video. Consistency builds association in your audience's mind.
3
Pin the product to your video when publishing ▼
  • On TikTok's publish screen, scroll down to Add Link
  • Select TikTok Shop Product → search for the product by name
  • The product appears as an orange button on the video — viewers tap and buy without leaving the app
  • Mention it out loud ("link is right on the video" or "tap the product below")
  • If exporting via CapCut directly to TikTok, you can add the product link right after
4
Track commissions and payment ▼
  • Creator Center → TikTok Shop → My Commissions
  • You'll see: product clicks, orders, conversion rate, amount pending
  • Payment: weekly, directly to your registered bank account
  • Holding period: commission is confirmed 7 days after product delivery to the customer
  • Track which videos drive the most sales → replicate the formats that convert

Module 9 · Content Structure

The Content Formula That Converts

Every successful affiliate video follows a 4-act structure. No exceptions.

01
HOOK — 0 to 3 seconds — The Question That Stops the Scroll ▼

If it doesn't stop the scroll in 3 seconds, the video doesn't exist for that user. Start with conflict, not with an introduction.

  • "Is this $90 serum actually worth it?" (doubt)
  • "I tested the most expensive Sephora product for 30 days…" (suspense)
  • "Why is everyone buying this at Ulta right now?" (FOMO)
  • "This product wrecked my skin… in the best way" (twist)
Rule: Never start with "Hey, how's it going?" or introducing yourself. The audience already knows who you are — they want to know what you have for them.
02
DEMO — 3 to 25 seconds — Real Demonstration ▼

Show application on skin, texture, scent (describe it), consistency. No hard cuts. No heavy filter. Authenticity is the product.

  • Before / after in one continuous clip
  • Zoom in on texture and pigmentation
  • Genuine real-time reaction
  • Talk about what you feel, not what the brand says
03
PROOF — 25 to 45 seconds — Concrete Result ▼

Numbers, comparisons, close-up skin shot, honest opinion. If it didn't work 100%, say so — that triples credibility.

  • "After 7 days, visibly smaller pores"
  • "Lasted 8 hours without a touch-up — I tested it at work"
  • "Didn't love the packaging, but the product itself…"
  • Mention price and where to buy at this point
04
CTA — Last 3 seconds — One Action, Only One ▼

Two CTAs = confusion = zero clicks. Choose ONE platform, ONE action, ONE link per video.

  • "Link in bio — buy directly at Ulta" (Ulta affiliate)
  • "Tap the product below" (TikTok Shop)
  • "Check my LTK — it's in the links" (LTK storefront)
  • "Comment WANT and I'll send you the link" (engagement + link in comments)
Why it works: "Comment WANT" drives comments → algorithm distributes more → more people see it → more sales. Use 2–3 times per week, not every video.

Module 10 · SEO

SEO by Platform

Each platform has its own search algorithm. The same content needs to be optimized differently for each one.

🎵 TikTok SEO
  • Spoken keywords: say the product name and category out loud in the first 10 seconds — TikTok transcribes and indexes it
  • On-screen text: put visible keywords in the video (e.g. "TULA Skincare Review")
  • Caption: first sentence with main keyword (not a hashtag — plain text)
  • Hashtags: 3–5 only — 1 broad (#beauty), 1 niche (#skincaeroutine), 1 specific (#tulareviews)
  • Trending sound: top-trending audios get automatic preferential distribution
  • Timing: post when your audience is active (Analytics → Followers → Activity)
▶️ YouTube SEO
  • Title (60 chars max): main keyword + benefit. Ex: "TULA Skincare Honest Review — 30 Days Results"
  • Description: first 150 chars are indexed — put keyword, product name, and affiliate link here
  • Tags: 10–15 tags, mix broad ("skincare review") and specific ("tula cleanser review 2026")
  • Chapters: add timestamps with keywords — they appear in Google results
  • Thumbnail: face + high contrast + 3 words max (e.g. "WORTH IT?")
  • Shorts: keyword in title + #Shorts in description + affiliate link in first 3 links
📸 Instagram SEO
  • Display name: include a keyword in your display name (e.g. "Vando | Skincare Reviews") — it appears in searches
  • Keyword-rich caption: Instagram indexes caption text — write naturally but include the product name and category
  • Alt text: on every post, edit the image alt text with descriptive keywords
  • Hashtags: 3–5 (the current algorithm penalizes 20–30 hashtags) — be specific
  • Reels description: first sentence with keyword — appears in Instagram Search
  • Location: geotag Nashville for local + national audience
👥 Facebook SEO
  • Groups: highest organic reach on Facebook — join beauty groups and post your videos with context
  • Native upload: always upload videos directly to Facebook (not YouTube links) — the algorithm favors native content
  • Video title: treat it like a blog headline — keyword + benefit + action
  • Description: first 3 lines appear without clicking "see more" — use keywords here
  • Alt text: set alt text on all product images
  • Page vs Profile: create a Creator Page for access to Facebook Insights and greater reach
📊
Universal cross-platform SEO rule: Same product, said in different ways, with keywords adapted for each platform. Never copy and paste the same caption to TikTok, Instagram, YouTube, and Facebook — each has its own search index.

Module 11 · Execution

90-Day Plan — From Setup to Income

Exact sequence of actions organized by week. Don't skip steps — each phase sets up the next.

1

Week 1 — Complete Setup

Before creating any content, configure your entire infrastructure.

  • Install Dolphin Anty + configure 2–3 profiles with residential proxies
  • Create TikTok accounts in Dolphin profiles — warm up for 3 days (consume, don't post)
  • Download CapCut (mobile + desktop) and Descript — create test projects
  • Open accounts on Impact.com, ShareASale, and LTK
  • Apply for: TikTok Shop Affiliate, LTK, Amazon Associates
  • Define the niche of each account — create optimized bios
  • Set up link in bio (LTK or Linktree) with affiliate links ready to go
2

Week 2 — First Videos + New Affiliates

Consistent posting starts here. Use the Hook→Demo→Proof→CTA formula in every video.

  • Post 1–2 videos per day on each active account (with TikTok Shop product pinned)
  • Record on smartphone → edit in Descript → finish in CapCut → publish
  • Apply for TULA Skincare (Impact.com) and Alpyn Beauty (ShareASale)
  • Use ChatGPT to generate hooks and scripts — test at least 3 types of openings
  • Reply to every comment within the first 30 min after each post
  • Do 2 Stitches of viral niche videos during the week
3

Week 3 — Ulta UBCreates + CapCut Affiliate

Add the slower-approval programs while your content is already running.

  • Apply for Ulta UBCreates (Rakuten / Impact) — process can take 1–2 weeks
  • Apply for CapCut Pro affiliate (Impact.com) — create 1 "how I edit my videos" video
  • Analyze early data: which hook type got the most watch time? Replicate it
  • Distribute top-performing videos to Instagram Reels and YouTube Shorts
  • Optimize SEO for each platform following Module 10
4

Weeks 4–8 — Consistency and Optimization

Learning phase. Data from the first weeks will show you what works.

  • Identify the 3 video formats that converted best → double production of those
  • Pause formats that got no results after 10+ videos
  • Test 1 new product per week on TikTok Shop — keep the ones that sell
  • Reach 1,000 followers on at least 1 account (TikTok Shop requires 1K)
  • Start a recurring series ("Weekly Skincare", "Find of the Week") — builds audience loyalty
  • Start collecting conversion data per affiliate program — cut the ones that don't convert
5

Month 2–3 — Scale What Works

Don't scale before you have data. With 30–60 days of data, you know what works.

  • Apply for Sigma Beauty (ShareASale) on the makeup account — when it has traction
  • Increase posting frequency in the best-performing niches
  • Outsource parts of the workflow: thumbnails, basic editing, calendar (use Fiverr freelancers)
  • Negotiate direct brand deals with brands you already promote organically
  • Reinvest first commissions into better proxies and more Dolphin profiles
  • Goal: first passive income from recurring affiliates (TULA subscriptions, CapCut Pro recurring)
